## Installation

```bash
npm i tailwindcss-plugins -D
```

If you're using Tailwind CSS version `0.x`, please run `npm i tailwindcss-plugins@^0.1.0 -D`.

## General usage

```js
plugins: [
    require('tailwindcss-plugins/PLUGIN_NAME'),
    // ...
],
```
